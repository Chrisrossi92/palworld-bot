FAQ

Common questions

Is PalMaster just another Palworld collection bot?

No. Collection features are part of the loop, but PalMaster is positioned around owner visibility, retention, and community engagement.

Is this free?

Yes during early access. Premium owner tools may come later, but Stripe and billing are not part of this phase.

Is it production-ready?

It is in active development and best for pilot/test servers. Use it when you are comfortable giving feedback during a soft launch.

What does PalMaster do today?

Players can use the current Palworld loop: starter rewards, captures, daily rewards, quests, Palbox/profile views, leaderboards, and public spawns.

What does PalMaster Dashboard show?

PalMaster Dashboard shows owner-facing activity such as total players, captures, owned Pals, daily quest activity, top collectors, Paldeck progress, and recent activity.

Does the dashboard require Discord login?

Yes. Discord OAuth verifies which installed servers you own or can manage.

Does installing the bot automatically grant dashboard access?

No. Bot installation and dashboard login are separate OAuth flows.

What happens after install?

Run /start, try /capture, claim /daily, check /quests, then return to the dashboard once players interact.

How do I get help?

Use the support link when configured. During early access, feedback and install issues should be routed through the support community.

Is billing available?

No. Billing and Stripe are intentionally out of scope for this phase.
